## 技术要点

### 百炼API调用
- 使用 dashscope Python SDK
- 调用 qwen-vl-plus 或同系列多模态模型
- 输入：书籍图片 + Prompt
- 输出：OCR文字（书名/作者/ISBN）+ 新旧程度评估

### 智能定价公式
```
最终价格 = 基础价格 × 新旧系数 × 版本系数 × 供需系数
```

### 环境变量配置
```
# config.py 中读取
DASHSCOPE_API_KEY=xxx  # 百炼API
MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=xxx
MYSQL_DATABASE=campus_books
```
