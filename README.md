# 校园二手书交易与智能定价系统
# Campus Second-Hand Book Trading & Smart Pricing System

## 项目简介
基于 Flask 的校园二手书交易平台，包含智能定价功能

## 技术栈
- 后端: Python 3.9 + Flask
- 前端: HTML5 + JavaScript + Bootstrap 5
- 数据库: MySQL

## 安装
```bash
pip install -r requirements.txt
```

## 配置
1. 修改 config.py 中的数据库配置
2. 初始化数据库: python init_db.py

## 运行
```bash
python run.py
```

## 功能模块
- 用户管理 (注册/登录)
- 书籍发布与检索
- 智能定价
- 交易撮合
- 订单管理
- 评价反馈
